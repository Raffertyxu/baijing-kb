# 白境空間清潔 — 小淨 AI 知識庫

官網右下角「小淨」聊天小幫手的問答資料。**改這裡的 `kb.json`,官網約 5 分鐘後自動生效**,不用動網站程式。

## 怎麼改

1. 點開 `kb.json` → 右上角鉛筆(Edit)
2. 照格式改內容 → 按 **Commit changes**
3. 等 5 分鐘,重新整理官網即可看到

## 格式說明

```json
{
  "faq": [
    { "q": "顯示在聊天室的常見問題按鈕", "a": "點了之後小淨的回答" }
  ],
  "kb": [
    { "k": ["觸發關鍵字1", "關鍵字2"], "t": "需求清單顯示名稱(可省略)", "a": "小淨的回答" }
  ]
}
```

- `faq`:聊天室下方的**常見問題圓形按鈕**
- `kb`:訪客**自己打字**時的比對規則
  - `k` — 訪客訊息只要包含其中任何一個詞就會觸發
  - `t` — 觸發後自動記進「需求清單」帶去 LINE 的服務名稱;純閒聊就不用填
  - `a` — 小淨逐字打出來的回答

## 注意

- **這份 kb.json 是小淨唯一的問答來源**,官網沒有備份——格式錯了小淨會暫時答不出問題(改回正確格式、等 5 分鐘即恢復)
- 標點、引號要用半形 `"`;改完先把內容貼到 https://jsonlint.com 驗證格式再 Commit 最保險
